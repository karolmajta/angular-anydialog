angular-anydialog

angular-anydialog allows you to use any dialog library with angular. Currently we support:

  • Boostrap 3
  • jQuery UI

If you want you can support whatever library you want, because you can always roll your own adapter!

It's super easy to start with!


                    <!-- the new and sexy <modal> tag is here to get you started -->

                    <modal name="yourModal"
                           controlled-by="yourModalController">
                           template-url="'path/to/your/modal-template.html'"></modal>

Check out the examples to see it in action.
