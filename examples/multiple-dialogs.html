This example shows how to handle multiple jquery-ui dialogs on one site. The related adapter is extremely simple, if you wish to use some more advanced jquery UI features (animations, dynamic title, variable number of dialogs etc.) you should probably roll your own.

After you show each dialog you can drag/resize it, just as you would use a normal jqueryUI dialog, but it can also be controlled with idiomatic angular code.
